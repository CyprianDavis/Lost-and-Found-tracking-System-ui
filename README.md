# AISMS-Dashboard

Simple React dashboard.

## Prerequisites

- Node.js 20.19+ (see `.nvmrc`)
- npm 10+

If you use `nvm`, run `nvm install` (if needed) followed by `nvm use` in the project root to pick up the pinned Node.js version.

## Development

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```
